Traka za privolu

CfC Consent

Code for Croatia traka za privolu je jednostavna implementacija za GDPR

Primjer upotrebe

Koristite traku za privolu kada pratite posjetitelje odnosno ako vaša stranica uz nužne kolačiće ima i one za svrhu prikupljanja i obrade osobnih podataka (marketinške, personalizacija, praćenje itd). Praćenje počinje tek nakon što posjetitelj da svoju privolu.

Kada se primijeni na vašu internetsku stranicu, Code for Croatia traka izgledat će ovako:

Demo

Kako koristiti

JavaScript

Dodajte JavaScript prikazan ispod u <body> vaše stranice te zamjenite 'UA-XXXXXX-XX' s vašim Google Analytics identifikatorom za praćenje:


          <script>
            var okiConsent = {
              analyticsTrackingID: 'UA-XXXXXX-XX'
            }
          </script>
          <script src="//cdn.jsdelivr.net/gh/codeforcroatia/assets@latest/assets/html/cfc/consent/assets/js/consent.js"></script>

Code for Croatia je neformalna inicijativa pozitivnih ljudi koji žele napraviti promjenu. Otključavamo zaključane podatke. Bavimo se civilnim hakiranjem, radimo društveno korisne projekte i uvijek smo otvoreni za nove ideje. Code for Croatia je križanje aktivizma, IT-a i istraživačkog novinarstva.
