Code for Croatia traka za kolačiće je jednostavna implementacija za EU zakon o kolačićima

Primjer upotrebe

Koristite traku za kolačiće kada ne pratite posjetitelje odnosno ako vaša stranica ima samo nužne kolačiće bez drugih svrha prikupljanja i obrade (marketinške, personalizacija, praćenje itd).

Kada se primijeni na vašu internetsku stranicu, Code for Croatia traka izgledat će ovako:

Demo

Kako koristiti

CSS

Dodajte kod u nastavku u <head> odjeljak svoje internetske stranice, kako biste uključili malu datoteku s definicijom stilova:

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/normalize/4.1.1/normalize.min.css"/>

JavaScript

Dodajte JavaScript prikazan ispod u <body> vaše stranice:

<script type="text/javascript"/>
              window.cookieconsent_options = {"message":"Ova internetska stranica koristi kolačiće. Za više informacija pogledajte našu ","dismiss":"U redu, shvaćam","learnMore":"politiku o kolačićima.","link":"https://codeforcroatia.org/privacy#cookie"};
          </script/>
          <script type="text/javascript" src="//cdn.jsdelivr.net/gh/codeforcroatia/assets@latest/assets/html/cfc/cookie-bar/js/cookie-bar.js"/></script/>

Code for Croatia je neformalna inicijativa pozitivnih ljudi koji žele napraviti promjenu. Otključavamo zaključane podatke. Bavimo se civilnim hakiranjem, radimo društveno korisne projekte i uvijek smo otvoreni za nove ideje. Code for Croatia je križanje aktivizma, IT-a i istraživačkog novinarstva.
